# VBForStatement
for文（.NET Core）

## 処理
1月から12月までを出力します。

## コード
```
Imports System

Module Program
    Sub Main(args As String())
        For i = 1 To 12
            Console.WriteLine(i & "月")
        Next
    End Sub
End Module
```

## 出力結果  
```
1月
2月
3月
4月
5月
6月
7月
8月
9月
10月
11月
12月
```
